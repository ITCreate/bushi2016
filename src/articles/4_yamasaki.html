シンプルなJavaScriptフレームワークVue.js

一年ほど前、いつまでもjQueryだけではダメだと思いAngularやReactを勉強しようと思いました。 しかしディレクティブやディスパッチャーなどわからない単語が多かったり、公式ページが英語だったりと壁が多く挫折してしまいました。 そんな時に出会ったのがVue.jsです。

Vue.js公式サイト(https://jp.vuejs.org/)

JavaScriptフレームワークの入門に最適！！

いつも周りの人におすすめしているのですが、Vue.jsの基本的な役割は双方向バインディングだけとシンプルなフレームワークなのでJavaScriptフレームワーク初心者の方でも学習しやすいです。 また公式ドキュメントの和訳がしっかりしているので英語が苦手な方でも大丈夫です。

表示したい部分には{{ 変数 }}、フォームにv-mode="変数" とするだけで双方向バインディングができます。 

import Vue from 'vue'

const app = new Vue({
    el: '#app',
    data: {
        msg: 'Hello!'
    }
})

<body>
    <div id="app">
        {{ msg }}
        <input type="text" v-model="msg">
    </div>
</body>

柔軟な拡張性

Vue.jsは双方向バインディングのサポートしかしてないのですが、公式ライブラリなど使用することで様々な規模のアプリケーションを作成することができます。

SPA

vue-routerを利用することで最近流行りのシングルページアプリケーションを作成することができます。

npm install vue-router

Fluxアーキテクチャ

規模が大きくなり複雑なアプリケーションになった場合はFluxライクなライブラリVuexを使います。

npm install vuex

テンプレート、スタイル、スクリプトをまとめてくれるvueファイル

Vue.jsでは.vueという拡張子のファイルを使用することができます。 コンポーネントの要素であるテンプレート、スタイル、スクリプトを一つのファイルにまとめることができます。 

<template>
    ここにテンプレート
</template>

<style scoped>
    ここにスタイル
    scopedをつけるとこのコンポーネント内だけで有効なCSSになります。
</style>

<script>
    ここにスクリプト
</script>

ただvueファイルを使用する場合はwebpackやbrowserifyを利用する必要があります。

コマンド一つでVue.jsを始める

初心者の壁になる一つがGulpなどのタスクランナーやwebpackなどのビルドツールです。 フレームワークを使って何か作りたいのにツールの設定方法などを勉強しないといけないのはもどかしいですよね。 Vue.jsではwebpackの設定やディレクトリ構成などをコマンド一つでしてくれるvue-cliというものがあるのですぐにVue.jsを使うことができます。 

npm install -g vue-cli

vue init webpack-simple#1.0 プロジェクト名

※webpack-simpleのあとに#1.0とつけない場合はvue2.0が入ります。

Vue.jsが新しく！

2016/10/1にVue.jsのバージョンが1.0から2.0になりました！ドキュメントの和訳はまだなのですが（11月中ぐらいには和訳されるらしい） Vue2.0では仮装DOMやサーバーサイドレンダリングなどが実装され速度の向上やSPAならではのSEO問題が解決されました。 またVue1.0との互換性がよくほぼそのままのコードでVue2.0を動かすことができます。
