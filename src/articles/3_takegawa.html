作品について

　作品は jsoup というjavaでHTMLの解析を行うライブラリと、
mecab という形態素解析エンジンを使用したwebスクレイピングなものが挫折してなければ展示されていると思います。
思い立った理由としては、「うんこ」って入力して「糞」って返って来たら面白いな〜って思ってました。疲れてるときに面白いと感じてしまったことを実行に移してしまい多少の後悔はあります。

使用言語

　今回javaのライブラリを使用してますが、scala で書いてます。 なぜscalaで書いているかと言いますと、部活の新入生歓迎会にて使わなくなった技術書をジャンケンで勝った人に譲るという場面がありました。ジャンケンする人は挙手参加型なのですが「なんかみんなこの本欲しそうだな〜」と思い適当に参加したら一人勝ちしてしまい、scalaの技術書（コップの本だよ！）を手に入れてしまいました。 顧問に「その本もらって勉強しなかったら先輩達の反感を買うことになるぞ」と言われたのでせっかく自由にプログラムを書けるいい機会なのでscalaで書くことにしました。

内容

　コードの内容としましては、jsoupさんにURLからHTMLを取ってきてもらい、そのHTMLから文字列を取得し、

def HTMLのやーつ: String = {
  //検索文字
    val goo = io.StdIn.readLine()
  //URLからHTMLを収納
    val doc: Document = Jsoup.connect("https:けんさくさいと" + goo + "+類義語").get()
  //bodyの文字列を取り出す
    val text: String = doc.body().text()
}

その後mecab様にお願いして解析してもらってあんなことやこんなことします（制作途中）
