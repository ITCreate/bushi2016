出展作品紹介

概要

今回出展させていただいたのは、 「 BookMark Binder 」という、Webアプリです。

これは、 「 お気に入りのリンクをファイルに出力するWebアプリ 」で、画面のフォームからお気に入りのリンクと付けたい名前を記述して登録するだけでファイルにお気に入りのリンクを出力することができます。

アプリによって出力されたファイルを画面から読み込むことでお気に入りを画面に表示させてお気に入りのリンクを利用できます。

機能一覧

  • お気に入りを登録する
  • お気に入りの一覧をファイルに出力する
  • ファイルからお気に入りの一覧を読み出すことが出来る

使用した技術

  • React.js
  • jQuery(アニメーション関係のみ)
  • JavaScript

なぜ作ったの？

常々思っていた、下記の問題を解決したいと思って作りました

  1. 複数のブラウザを使う身からするとお気に入りが共有できない
  2. 勉強した時に使ったWebサイトを まとめて共有・保存 しておきたいときにもう一度リンクを探すのが大変
  3. ナイショのお気に入り とかを作りたいときに家族共有PCとかだと作れない

どうやって解決したか

  • 1 => ファイル化するので、各ブラウザでお気に入りファイルを読みこめば良い
  • 2 => 開いていたタブのリンクをファイルに出力してそのファイルを Slack や GitHub などで共有するようにすればOK
  • 3 => 出力したファイルを誰にも見られないところに保存しておけばOK…

これから追加したい機能

  • お気に入りを編集する機能
  • お気に入りを削除する機能
  • お気に入りに対してコメントを付ける機能

詰まった点・苦労した点

  • React の jsx の記法に悩まされました…
  • この this(props) は何が入ってるんや…
